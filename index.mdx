# Global Lirmi Demo App

## ¿Qué hace?
Esta aplicación es un demo basado en Blazor WebAssembly para la plataforma Lirmi, mostrando arquitectura moderna y buenas prácticas.

## ¿Cómo se usa?
Sigue la guía de inicio rápido en `/pages/getting-started.mdx` para clonar, compilar y ejecutar el proyecto localmente.

## Dependencias
- .NET 10 SDK
- Visual Studio 2026 o superior
- Blazor WebAssembly

## Ejemplo
Consulta la sección de componentes en `/components/components.mdx` para ver ejemplos prácticos de uso de componentes Blazor.
