# Guía de inicio rápido

## ¿Qué hace?
Explica cómo clonar, compilar y ejecutar la aplicación Blazor WebAssembly localmente.

## ¿Cómo se usa?
1. Clona el repositorio:
   ```bash
   git clone https://dev.azure.com/lirmi/lirmi.net/_git/global-lirmi-demo-app
   ```
2. Abre la solución en Visual Studio 2026 o superior.
3. Restaura los paquetes NuGet y compila la solución.
4. Ejecuta el proyecto `LirmiDemoApp.Client` (Blazor WebAssembly).

## Dependencias
- .NET 10 SDK
- Visual Studio 2026 o superior

## Ejemplo
Consulta `/components/components.mdx` para ejemplos de componentes Blazor en acción.

## Requisitos
- .NET 10 SDK
- Visual Studio 2026 o superior

## Estructura básica
- `LirmiDemoApp.Client`: Proyecto principal Blazor WebAssembly
- `LirmiDemoApp.Server`: Backend opcional (si aplica)
